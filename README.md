# ProjectHelpDesk

Frontend : Angular 6

Backend: Api utilizando REST, JWT, Spring Boot, Security, Data e MongoDB

Recursos:

* Incluir, Editar, Excluir e Listar usuários.
* Incluir, Editar, Excluir e Listar tickets.
* Resumo com as informações sobre os tickets.
